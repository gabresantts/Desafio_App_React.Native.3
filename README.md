# Desafio_App_React.Native.3
Desafio sobre um app React Native com Expo que realize operações de CRUD com um Back-End. O objetivo é conseguir cadastrar, ler, editar e deletar dados de uma entidade. 

# Desafio 3 - CRUD com Expo SDK 54

Aplicativo React Native com Expo SDK 54 e backend Node.js para operações CRUD.

## Funcionalidades

- Listar usuários
- Criar novos usuários
- Editar usuários existentes
- Excluir usuários
- Visualizar detalhes

## Como Executar

### Backend
```bash
cd backend
npm install
npm run dev
